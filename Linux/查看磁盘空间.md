# Linux查看磁盘空间(df, du)

> 以下命令可以查看磁盘各分区大小、已用空间等信息：

```shell
df -h
```

> 以下命令可以查看foo目录的大小：
```shell
du -sh foo
```

> 有时候，硬盘比较满了，我们想找一些目录来清除，可以用下面命令查看当前目录以下搜索文件和子目录大小。找出特别大的，看里面有没有文件可删：
```shell
du -sh *
```

> 如果我们插入了一个U盘或移动硬盘，可以用df命令查看它挂载的地方，通常在/mnt或/media下。如果想卸载USB存储设备，可以用umount命令：
```shell
umount path
```
以上命令更多参数可查看man或info。
